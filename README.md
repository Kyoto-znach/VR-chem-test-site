# Прототип сайта лабораторных работ

Этот репозиторий содержит ранние версии и прототипы сайта для размещения лабораторных работ студентов. Проект создан в рамках учебной практики и является частью моего обучения веб-разработке.

## О проекте

Цель — создать удобную и простую платформу, где можно:

- загружать лабораторные работы;
- оформлять их по шаблону;
- удобно просматривать и редактировать.

##  Что внутри

Репозиторий включает несколько версий сайта, сделанных на HTML, CSS и JavaScript. Некоторые из них были тестовыми и не пошли в основную разработку, но выложены как пример моего прогресса.

## Технологии

- HTML5
- CSS3
- JavaScript (основы)
- [React (в будущем планируется)]()
